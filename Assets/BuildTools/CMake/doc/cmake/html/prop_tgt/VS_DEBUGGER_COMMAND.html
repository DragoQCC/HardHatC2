VS_DEBUGGER_COMMANDÂ¶

New in version 3.12.

Sets the local debugger command for Visual Studio C++ targets. The property value may use generator expressions. This is defined in <LocalDebuggerCommand> in the Visual Studio project file. This property is initialized by the value of the variable CMAKE_VS_DEBUGGER_COMMAND if it is set when a target is created.

This property only works for Visual Studio 12 2013 and above; it is ignored on other generators.
